# Task 6: Data Visualization – Python Charts for Insights

## Tools Used
- Google Colab
- Python
- pandas, matplotlib

## Dataset
Retail sales dataset created and used for visualization.

## Visualizations
- Bar Chart: Product-wise sales
- Line Chart: Monthly sales trend
- Histogram: Profit distribution
- Scatter Plot: Sales vs Profit

## Insights
1. Certain products contribute significantly to total sales.
2. Sales trends over time show peak and low periods.
3. Sales and profit show a positive correlation.
