# Task 3 – MySQL Data Import

## Description
This task demonstrates data cleaning, MySQL table creation, and CSV data import using MySQL Workbench.

## Steps Performed
- Cleaned dataset in Excel
- Saved cleaned data as CSV
- Created MySQL database and table
- Imported CSV using Table Data Import Wizard
- Verified data using SELECT queries

## Tools Used
- MySQL Server
- MySQL Workbench
- Microsoft Excel
